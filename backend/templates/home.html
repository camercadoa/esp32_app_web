Sistema desarrollado con ESP32 y una interfaz web para monitorear, controlar y analizar las acciones de un motor DC y un conjunto de LEDs.

Acerca del Proyecto

Este proyecto integra un microcontrolador ESP32 con una aplicación web que permite:

  • Encender, apagar y controlar la dirección de un motor DC.
  • Controlar y monitorear un conjunto de LEDs.
  • Registrar automáticamente las acciones ejecutadas por el usuario.
  • Visualizar estadísticas y generar reportes de las acciones.

Todo esto con una interfaz moderna, adaptable y conectada mediante APIs REST.

Módulo de Control

Permite encender o apagar el motor DC, cambiar su dirección de giro y controlar los LEDs desde el navegador.
